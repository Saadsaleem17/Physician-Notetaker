# Sample Output - README

This directory contains the output files generated by the Medical NLP Pipeline.

## Files

- `analysis_results.json` - Complete analysis results (generated after running main.py)

## Structure

```json
{
  "entities": {
    "Patient_Name": "Janet Jones",
    "Date_of_Incident": "September 1st",
    "Symptoms": [...],
    "Diagnosis": "...",
    "Treatment": [...],
    "Current_Status": "...",
    "Prognosis": "..."
  },
  "keywords": [...],
  "json_summary": {...},
  "sentiment_intent": [...],
  "soap_note": {
    "Subjective": {...},
    "Objective": {...},
    "Assessment": {...},
    "Plan": {...}
  }
}
```

## Usage

Run the main application to generate output:

```powershell
python main.py
```

The results will be saved to `output/analysis_results.json`.
